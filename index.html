Ideas Principales

  • Reglas Generales de Especificidad:ID tiene la mayor prioridad, seguido de clases, atributos y pseudo-clases, y finalmente etiquetas o elementos. Los ID siempre tienen un valor de especificidad más alto que las clases o las etiquetas. Los selectores de clase (como .mi-clase) y pseudo-clases (como :hover, :nth-child) tienen más peso que los selectores de tipo (como div, p). Un ID supera a cualquier cantidad de clases, atributos o elementos.
  • Reglas de Importancia y !important: El uso de la regla !important puede alterar la especificidad. Aunque una regla con !important no se calcula según la especificidad normal, tendrá prioridad sobre las reglas sin !important, sin importar la especificidad de estas últimas.
  • El Orden de las Reglas en Caso de Igualdad:Si dos reglas tienen la misma especificidad, la que aparece última en el código CSS prevalecerá.
  • Especificidad en los Selectores Compuestos:Cuando se usan selectores compuestos (combinando elementos, clases y atributos), el puntaje se suma.

Apuntes de Clase

  • Especificidad Css La especficidad es el medio por le cual los navegadores deciden que valores de propiedad Css son los mas relevantes para un elemento y, por lo tanto, se aplicaran.
  • Como se calcula ?La especificidad solo se aplica cuano el mismo elemento es el objetivo de varias declaraciones. Segun las relgas de Css, los elementos dirigidos directamente siempre tendran prioridad sobre las reglas que un elemento hereda de su antepasado.
  • Clases, atributos y pseudo-clases Clases, atributos y pseudo-clases : esta categoria incluye, clases, atributos y pseudoclases como: hover, focus, etc.

Resumen

La especificidad en CSS3 es un sistema que determina qué reglas se aplican cuando varias coinciden con un elemento. Se calcula con base en la combinación de selectores de ID, clase, atributos, y elementos. Los ID tienen mayor prioridad que las clases, y estas a su vez tienen más peso que los elementos. El uso de !important puede alterar este sistema, otorgando prioridad a la regla marcada como importante.
